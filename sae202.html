SAE 2.02 : Mesurer et caractériser un signal

Description du projet

Cette SAE consiste à créer une web radio connecté à partir d'une carte électronique ESP32 et un script Arduino. Une fois la web radio fonctionnel, on peu l'utiliser via une application mobile que l'on crée avec IOT MQTT Panel. Cette application est directement liés avec le script via un partage de connexion et l'application envoie les données (chaine de radio, réglage volume, tonalité et spacialisation) à un topic mqtt.
