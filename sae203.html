SAE 2.03 : Solution informatique pour l’entreprise

Description du projet

L'objectif de cette SAE est de développer une application web pour une entreprise, dans mon cas une compagnie aérienne. Ce projet en groupe de 4 commence par une gestion de projet afin de prévoir la répartitions des tâches que ça soit entre équipe ou dans le temps. Cela demande ensuite toute une réflexion dans l'organisation de la base de données (externe à l'application), en effet, il faut trouver une bonne logique de base de données qui permette la gestion du trafic aerien. Puis il y a la partie la plus longues "le développement du projet django", créer des actions de CRUD dans views.py, gérer les liens dans urls.py, créer des templates html et de l'afichage css etc... Et bien sur finir avec une soutenance accompagné d'un support de présentation.
