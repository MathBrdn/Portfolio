SAE 3.04 Dev Cloud : Mettre en place une infrastructure virtualisée

Description du projet

L'objectif de cette SAE est de répondre à un appel d'offre. Une petite patisserie souhaite rendre son site e-commerce, plus performant en période de rush. Le but est d'étudier la problématique de cette entreprise dans le but leur proposer une offre cloud qui correspond à leurs attente. Ainsi il faut rédiger un rapport contenant une introduction, un résumé managérial, l'explication de notre offre (nos choix techniques), évoqués les coûts qu'implique cette offre, proposer quelques alternatives etc... Il faut également réaliser un support de présentation, puis passer une soutenance dans laquel, on explique nos choix.
